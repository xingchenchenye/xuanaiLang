# 变更记录

## 0.7.0

- 新增结构体系统，支持 `构 类型 { ... }` 与 `新 类型 { ... }`
- 支持结构字段读取、结构字段赋值和结构嵌套使用
- 三条执行链全部支持结构体：解释器、Python 后端、VM 后端
- 反汇编能力新增 `STRUCT` 与 `STRUCT_INIT`
- 增加结构体示例与专项测试
- 新增异常处理系统，支持 `试 / 捕 / 抛`
- 断言失败现在可以在语言内被捕获和恢复
- 运行时错误可以在捕获分支中以结构化错误载荷读取
- 解释器、Python 后端、VM 后端三条执行链全部支持异常流
- 增加异常演示示例与专项测试

## 0.6.0

- 新增复杂左值赋值能力
- 支持 `目标.字段 = 值`
- 支持 `目标["键"] = 值`
- 支持成员和索引混合嵌套赋值
- 三条执行链全部支持复杂左值

## 0.5.0

- 新增原生映射系统
- 支持映射字面量 `{"键": 值}`
- 支持映射类型 `映[键类型, 值类型]`
- 增加 `标准.数据` 模块与映射相关能力
- 三条执行链全部支持映射构造与索引

## 0.4.0

- 新增独立 VM 字节码执行后端
- 新增字节码编译器、VM 执行器与反汇编能力
- 模块导入与缓存机制支持多后端一致运行
- 增加 VM 后端专项测试

## 0.1.x - 0.3.x

- 建立语言基础内核
- 完成词法、语法、AST、语义分析和基础优化
- 建立 Python 后端、CLI 工具、示例和测试体系
- 接入张量与 AI 运行时原型
